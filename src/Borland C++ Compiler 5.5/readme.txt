・このプログラムについて

　入力された自然数を素因数分解したり、指定範囲で素数を探すシンプルなプログラムです。
　しかし、その裏には計算中の"応答なし"回避のためのマルチスレッド化や計算結果のエディットボックス表示その他のWindows APIを用いたC++のGUIプログラミングにおける基礎的な内容をたくさん詰め込んでいます。GUIアプリケーションのテンプレートとしても活用できるかと思います。
　DXライブラリを使えばもっと簡単に作れますが、Windows APIに挑戦してみたいという私のような変わり者はぜひ参考にしてみてください。


・ビルド時の要件

　このプロジェクトは、BCC Developerで作成されています。
　また、コンパイラはBorland C++ Compiler 5.5でのみ動作確認がとられています。
　なお、このコンパイラは超古いのでWindows XPにも普通に対応している他、64bitプログラムのビルドは行えません。


・注釈

　ソースコードは、このファイルと同階層にある1.cppです。
　実行可能バイナリは、デバッグビルドはDebugフォルダ、リリースビルドはReleaseフォルダ
　にPF_IA-32.exeとして生成されます。
